# 经营链条验证：6问题方法论

> 本框架的核心思想：**利润可以调节，但现金流动和资产堆积难造假**。
> 6 个问题串起完整的「订单 → 现金 → 产能 → 扩张」闭环，每一环都用资产负债表或现金流量表的数据，去交叉验证利润表讲的故事。

---

## 完整链条全景

```
  Q1 订单真实       Q2 利润现金含量       Q3 现金真的来了
  （合同负债）       （经营现金流/净利润）   （销售收现/营收）
       ↓                    ↓                     ↓
                  Q4 应收未恶化（议价能力）
                                ↓
              Q5 产能扩张（账面 + 现金双视角）
                                ↓
                  Q6 扩张安全性（有息负债+财务费用）
```

每个问题对应 1-3 个具体指标，下面逐一展开。

---

## Q1. 客户真下单了吗？——看合同负债

### 指标
**合同负债**（资产负债表科目）= 客户预付给公司但货还没发的钱。

### 为什么重要
合同负债是**业绩的先行指标**——它说明订单已经转化为客户的真金白银，不是 PPT 上的意向。比券商分析师的预测靠谱得多。

### 怎么算/怎么看
- 比较：**本期末 vs 上期末**（环比）
- 信号判定：
  - 🟢 增长 > 5%：订单池在扩，业绩有支撑
  - 🟡 ±5% 区间：业务平稳
  - 🔴 下降 > 10%：订单池萎缩，要警惕未来收入

### 必须知道的一个常识

**合同负债 vs 预收款项**：2017年新收入准则后，A股报表的"预收款项"科目已被拆分到**合同负债**（与销售合同直接对应）和**其他流动负债**（如代收款项）中。如果在新版报表里看不到"预收款项"行不要慌，看合同负债就对了。

### 行业差异
- 制造业（设备、工程）：合同负债通常占年度营收 5%-30%，是核心指标
- 消费品：周转快、合同负债意义弱化，重点看应收和存货
- 平台/SaaS：递延收入（合同负债的另一种表现）= 已收钱未交付服务，意义类似
- 零售：基本无合同负债概念，跳过此题

## Q2. 利润有现金撑住吗？——看经营现金流 ÷ 归母净利润

### 指标
**利润现金保障倍数** = 经营活动产生的现金流量净额 ÷ 归属于母公司股东的净利润

### 为什么重要
这是验证利润真实性的**第一把钥匙**。利润表上 1 块钱利润，背后是不是有 1 块钱现金支撑？如果不是，那利润可能就是"纸面富贵"。

### 怎么算/怎么看
- 比较：本期 vs 上年同期 vs 全年（至少看 3 个时段）
- 信号判定：
  - 🟢 长期 > 1：利润质量优秀，每块利润都有现金支撑
  - 🟢 0.8 - 1：利润质量正常
  - 🟡 0.3 - 0.8：利润有水分，警惕应收账款增长
  - 🔴 < 0.3 或为负：利润质量差，可能是纸面富贵

### 特殊情形（必须掌握）

**当净利润为负时**：直接用比值意义不大，要换成**绝对值对比**——
- 经营现金流 > 净利润亏损额 → 现金流好于账面亏损 → 亏损是费用前置/收入确认滞后导致的"账面亏"，**积极信号**
- 经营现金流 < 净利润亏损额 → 现金流比账面亏损还差 → 公司在真金白银流血，**警告信号**

**当净利润为正、经营现金流为负时**：极度警惕。可能是大规模赊销支撑利润，应收账款会暴雷。

---

## Q3. 钱真收回来了吗？——看销售收现 ÷ 营收

### 指标
**销售变现率** = 销售商品、提供劳务收到的现金 ÷ 营业收入

### 为什么重要
这是验证利润真实性的**第二把钥匙**。营收是会计概念（确认了就算），销售收现是真金白银。两者的比值反映"销售变现能力"。

### 怎么算/怎么看
- 注意：销售商品收到的现金**包含增值税**，营收**不含税**，所以健康公司的比值应在 **1.10 - 1.17**（按 13% 增值税率粗算）
- 信号判定：
  - 🟢 > 1.15：销售变现优秀（不仅当期收齐，还在收历史欠款 / 大量预收新订单）
  - 🟢 1.00 - 1.15：正常
  - 🟡 0.85 - 1.00：有部分赊销，需结合应收变化看
  - 🔴 < 0.85：大量收入挂在应收账款上，存在坏账风险

### 与 Q2 的关系
Q2 看"利润 vs 现金"，Q3 看"营收 vs 现金"。两者一起看才完整：
- Q2 高 + Q3 高 → 利润和营收都有现金支撑，最理想
- Q2 低 + Q3 高 → 营收变现良好，但成本/费用吃掉了利润（看 Q5 是否在扩张）
- Q2 高 + Q3 低 → 罕见，可能是非经常性损益拉高利润
- Q2 低 + Q3 低 → 警告

---

## Q4. 应收账款恶化了吗？——看增长时的回款质量

### 指标
**应收账款 + 销售收现/营收 + 合同负债** 三者联动

### 为什么重要
这是验证客户真实性和议价能力的**关键交叉点**。一家健康公司，收入增长时不应该让应收账款同步膨胀（那样意味着客户在拖欠）。

### 怎么算/怎么看
- 看**应收账款增速 vs 营收增速**的对比
- 信号判定：
  - 🟢🟢 营收↑ + 应收↓ → 极佳信号（议价力强，客户付款积极）
  - 🟢 营收↑ + 应收同比例↑ → 正常增长
  - 🟡 营收↑ + 应收增速 > 营收增速 → 需警惕（开始放宽信用换收入）
  - 🔴 营收持平 + 应收暴增 → 大概率是纸面收入

### 完美组合
当出现 **"营收↑ + 应收↓ + 销售收现/营收 > 1 + 合同负债↑"** 这四联动时，几乎可以断定：客户结构发生了质变（要么换了高质量客户，要么议价权大幅提升）。

### 行业差异
- 设备制造：应收账款占营收 50%-100% 都正常（账期长）
- 消费品：应收账款占营收 < 20%
- 工程承包：合同资产 + 应收账款都要看
- 平台型：应收账款通常很低，重点看 GMV 和现金循环周期

## Q5. 产能扩张是真金白银吗？——账面 + 现金双视角验证

### 指标（两套数据交叉看）

**账面视角（资产负债表）**：四个科目的递进关系
- 预付款项（备料）→ 存货（在产）→ 在建工程（在建产能）→ 固定资产（已落地产能）

**现金视角（现金流量表）**：
- **资本开支** = 购建固定资产、无形资产和其他长期资产支付的现金

### 为什么重要
账面增长和现金支付**两套数据交叉**才完整：
- 账面增长但现金没付 → 可能是收购并表带入，不是新投入
- 现金付了但账面没增加 → 钱在路上（设备未到货 / 折旧抵减 / 汇率折算），**未来 1-2 个季度会陆续转固落地**
- 两者同步增长 → 健康扩张

### 怎么算/怎么看

**账面四线**：分别看本期末 vs 上期末的环比变动
- 🟢🟢🟢 预付款项暴增（+50%以上）：备料生产，是订单饱满的最强信号
- 🟢 存货稳健增长（10%-30%）：为后续交付做准备
- 🟢 在建工程增长：产能扩张"水龙头"还开着
- 🟡 固定资产小幅波动：受折旧和汇率折算影响，不是产能信号

**资本开支**：
- 比较：本期 vs 上年同期，以及 **资本开支占营收比**
- 信号判定：
  - 🟢🟢 占营收 > 20%：激进扩张，预示新一轮增长周期
  - 🟢 占营收 5%-20%：正常投入
  - 🟡 占营收 < 5%：维持性投入，扩张意愿弱
  - 🔴 资本开支断崖式下降：可能放弃扩张

### 必看的"差额信号"
计算：**资本开支现金支付额 - (在建工程账面增量 + 固定资产账面增量)**
- 差额接近 0：账面与现金对得上
- 差额很大（比如现金 5,000 万，账面只增 100 万）：**大量产能在途，未来 1-2 季度会陆续转固落地** —— 这是个"埋雷式"的积极信号

### 行业差异
- 重资产制造业：四线全看
- 轻资产/消费/平台：固定资产意义弱，重点看**研发投入**（计入费用）和**门店/SaaS 客户数**等业务指标
- 科技公司：除了固定资产还要看**无形资产 + 开发支出**

## Q6. 扩张会不会玩火？——看有息负债 + 财务费用

### 指标
**有息负债** = 短期借款 + 长期借款 + 一年内到期的非流动负债 + 应付债券
**财务费用率** = 财务费用 ÷ 营业收入
**资产负债率** = 总负债 ÷ 总资产

### 为什么重要
扩张是好事，但**过度加杠杆就是玩火**。很多公司倒在扩张得太快、被利息支出拖垮上。

### 怎么算/怎么看

**有息负债环比**：
- 🟢 < 5%：可控
- 🟡 5%-15%：扩张中，需要持续盯
- 🔴 > 15%：激进加杠杆，风险升高

**财务费用率**（核心指标）：
- 🟢 < 3%：很轻松
- 🟡 3% - 5%：制造业舒适区
- 🔴 > 5%：偏高，开始侵蚀利润
- 🔴🔴 > 8%：警戒，已经是利润杀手

**资产负债率**（行业差异大）：
- 制造业：50%-65% 正常
- 消费品：30%-50% 正常
- 房地产 / 建筑：70%-85% 正常但需结合现金流看
- 科技 / 平台：< 40% 健康

### 与 Q5 的关联
扩张速度越激进（Q5 信号越强），就越要看 Q6 是否安全。一个理想的扩张应该是：**资本开支放量 + 有息负债增速可控 + 财务费用率不破警戒线**。

```
  ①合同负债 [+/- X%]    ②利润现金保障 [比值]    ③销售收现 [Y%]
      订单[真/弱]            利润质量[优/差]        现金流入[强/弱]
         ↓                      ↓                      ↓
                  ④应收账款 [+/- Z%]（营收 [+/- W%] 下）
                       议价力 [强/弱]
                              ↓
              ⑤产能扩张：账面四线 + 现金 X 万/季
                  预付/存货/在建/固定 各自变动
                  现金 vs 账面 差额 → 在途产能信号
                              ↓
                  ⑥有息负债 X 亿（[+/- Y%]） [✅/⚠️]
                     扩张[安全/玩火]
```

## 总判断模板

完成 6 题后，给出一句话总判断：

**"X 个积极信号 + Y 个警告信号 + Z 个待观察。经营链条整体[健康/有隐忧/恶化]，[最大亮点是 XX]，[最大风险是 XX]。"**

例如：
- "5 个积极信号 + 1 个警告信号。经营链条整体健康，最大亮点是订单+回款质量双高，最大风险是有息负债偏紧。"
- "2 个积极信号 + 3 个警告信号 + 1 个待观察。经营链条已显疲态，最大风险是利润现金含量持续低于 0.3，需警惕暴雷。"
